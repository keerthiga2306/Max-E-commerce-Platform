# 🛒 Max Online E-Commerce Database Management System

## 📖 Introduction

The **Max Online E-Commerce Database Management System** is a web-based application designed to simplify and automate online shopping operations. It provides customers with a secure platform to browse products, add items to a shopping cart, place orders, make online payments, and track purchases.

The system also enables administrators to manage products, categories, customers, orders, inventory, and sales reports through a centralized database. The project is developed using **HTML, CSS, JavaScript, PHP, and MySQL**.

# ▶ Installation

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/max-online-ecommerce.git
```

### 2. Move Project

Copy the project folder into

```
xampp/htdocs/
```

### 3. Import Database

- Open phpMyAdmin
- Create database

```
max_online_ecommerce
```

- Import

```
database/ecommerce.sql
```

### 4. Configure Database

Edit

```
config.php
```

```php
$host = "localhost";
$user = "root";
$password = "";
$database = "max_online_ecommerce";
```

### 5. Run Project

Start:

- Apache
- MySQL

Open:

```
http://localhost/max-online-ecommerce/
```
